# 商品管理システム

## 概要
このシステムでは、店舗で扱う商品の在庫管理を行うことが出来ます。
商品の登録、編集、削除を行うことができ、在庫の確認と在庫数の変更を行うことが出来ます。

## 主な機能
- ログイン、ログアウト機能
- 商品一覧画面
- 商品登録、編集、削除機能
- 商品検索機能
- 商品在庫一覧画面、在庫管理機能 

## 開発環境

```
PHP 7.4
MySQL 5.6
Laravel 8.83
```
